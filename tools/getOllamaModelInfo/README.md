# Get Ollama Model Info

**getOllamaModelInfo** - A tool for the [Ollama Bash Toolshed](../../).

Get info about an Ollama model. 

* Model info
  * architecture
  * parameters
  * context length
  * embedding length
  * quantization
* Capabilities
* Parameters
* License
* (other info possible, per model)

# Parameters

- ```model```

# Requirements

- ```ollama``` command
