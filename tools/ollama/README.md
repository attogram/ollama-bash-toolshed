# ollama

**ollama** - A tool for the [Ollama Bash Toolshed](../..).

# scripts
- ```list.sh```
- ```ps.sh```
- ```show.sh```
- ```capabilitites.sh```
- ```capable.sh```

# Examples running locally
- ```/exec ollama action=version```
- ```/exec ollama action=list```
- ```/exec ollama action=show model=qwen3:0.6b```

# Parameters

- ```action``` (required)
- ```model``` (required for action=model_info)

# Requirements

- ```ollama``` command
